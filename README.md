<p align="center" style="font-size:18px">
A Java-based automation system that integrates Shoper, CRM, 3uTools diagnostics, and FakturaXL invoicing.
<br>
Automates order processing, device analysis, PDF label generation, CRM communication, and invoice workflows.
</p>

---

<h2>Features</h2>

<ul>
  <li><b>Shoper Integration</b>: Order fetching, parsing, and auto-comments</li>
  <li><b>CRM Automation</b>: Device submission and problem handling</li>
  <li><b>3uTools Diagnostics</b>: Device model, IMEI, capacity, color detection</li>
  <li><b>PDF Labels</b>: HTML-based printable logistics labels</li>
  <li><b>FakturaXL Automation</b>: Login/session management</li>
  <li><b>UI Components</b>: Login view, orders, device tools, invoices</li>
</ul>

---

<h2>Tech Stack</h2>

<p>
<b>Java 17</b>, Maven, JSoup, HttpClient, Gson/Jackson, OpenPDF, FlyingSaucer, Lombok, SLF4J/Logback
</p>
